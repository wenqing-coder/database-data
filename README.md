# mysql-demo-data

#### 介紹
这是MYSQL官方提供的数据库样本，用于测试你的应用程序和数据库服务器

#### 要求
MySQL数据库服务器(5.0以上)

#### 安裝教程

1. git clone 仓库 / 下载zip包
2. cd 下载好的仓库目录下

#### 使用说明

1. mysql -u root -p < employees.sql

2. mysql -uroot -p < test_employees_sha.sql(校验是否安装成功)
